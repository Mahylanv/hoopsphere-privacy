# Politique de confidentialité — Hoopsphere

Dernière mise à jour : 28 janvier 2026

Hoopsphere respecte votre vie privée. Cette politique explique quelles données sont collectées et comment elles sont utilisées.

## Données collectées
Selon votre utilisation de l’application, nous pouvons collecter :
- **Compte utilisateur** : email, nom/prénom, photo de profil, informations sportives.
- **Contenu publié** : photos, vidéos, posts, commentaires.
- **Données de paiement** : les paiements sont traités par **Stripe**. Hoopsphere ne stocke pas vos informations bancaires.
- **Données techniques** : type d’appareil, système, logs en cas d’erreur.
- **Permissions** :
  - **Caméra** : utilisée pour prendre ou importer une photo/vidéo de profil ou de publication.
  - **Galerie/stockage** : utilisée pour choisir des images/vidéos.
  - **Notifications** (si activées) : pour vous notifier d’activité.

## Utilisation des données
Les données sont utilisées pour :
- créer et gérer votre compte,
- afficher votre profil et vos contenus,
- permettre l’interaction entre clubs et joueurs,
- gérer les paiements et abonnements,
- améliorer l’application.

## Partage des données
Nous ne vendons pas vos données.  
Certaines données peuvent être partagées uniquement avec :
- **Stripe** (paiement),
- **Firebase/Google** (authentification, base de données, stockage).

## Sécurité
Nous mettons en place des mesures techniques pour sécuriser vos données.

## Vos droits
Vous pouvez demander la suppression de votre compte et de vos données.
